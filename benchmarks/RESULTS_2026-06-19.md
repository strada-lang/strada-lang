# Benchmark sweep — 2026-06-19

Five full multi-language benchmarks across all six languages. Wall-clock
seconds, **best of 3 runs**, lower is better. Run via
`RUNS=3 ./run_benchmarks.sh bench_compute bench_functions bench_oop bench_strings bench_array_hash --langs strada,perl,python,ruby,php,node`.

- Box: 2-core AWS dev host, light load.
- Toolchains: Strada (this tree) / Perl 5.38.2 / Python 3.12.3 / Ruby 3.2.3 /
  Node v24.15.0 / PHP 8.3.6.

## Notes

- Strada wins every benchmark against every language.
- **Strings vs Python/Ruby** (2524× / 1102×) is a known pathology, not a
  general result: those benchmarks build strings with `+=` on immutable
  strings (O(n²) copies). The fair string margin is the ~1.3–2.6× seen vs
  Perl/Node/PHP.
- **OOP improved ~2×** since the prior recorded numbers (Strada 0.051s →
  0.023s), roughly doubling the OOP speedups (e.g. vs Perl 30× → 56×).
- Node (V8 JIT) is the closest competitor (1.9–5.4×); the interpreted
  languages (Perl/Python/Ruby) trail by 1–2 orders of magnitude on
  compute/function/OOP work.
